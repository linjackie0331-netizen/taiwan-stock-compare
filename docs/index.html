📖 DuPont 公式說明

  • ROE = 淨利率 × 資產周轉率 × 財務槓桿倍數
  • 淨利率高 → 獲利能力強（品牌/技術護城河）
  • 資產周轉率高 → 資產使用效率高（零售/製造業優勢）
  • 財務槓桿高 → 借款放大獲利（金融業常見，注意風險）

⚖️ 金融檢察官在找什麼？

  • 盈餘品質：淨利 vs 現金流是否吻合？（不吻合 = 可能帳面造假）
  • 有盈無現：帳面賺錢但持續無現金 = 台灣掏空案最典型特徵
  • 資產膨脹：資產成長遠超營收 = 可能用關聯交易虛增資產
  • 應收帳款暴增：塞貨給通路或虛開發票（Channel Stuffing）
  • 毛利率突跳：費用資本化、成本操縱
  • 股東權益流失：利潤無法解釋的消失 = 疑似掏空

🔴 股東權益異常流失（疑似掏空）
近 5 年累計淨利約 22,870 億，但股東權益僅從 12,264 億變為 19,298 億（應有 35,134 億），差距 15,837 億（129%）無法以股利完全解釋。經典掏空手法：以高額股利為名分配現金後再增資圈錢、或透過關聯方交易將資金轉至私人帳戶。
🔍 可疑電子B（示範：多項警示） 財報偵查報告
🔴 盈餘品質異常
連續 5 年 OCF/淨利 < 80%（平均 42%）。淨利遠超現金流是財報操縱最典型特徵，常見手法：提前認列收入、費用遞延至下一期、應收帳款虛增。
🔴 資產異常膨脹
資產年均成長 26.0%，遠超營收成長 5.0%（差 21.0ppt）。常見手法：用高價向關係人購入資產、虛增存貨或預付款項、投資空殼子公司將現金轉出。
🔴 應收帳款暴增（Channel Stuffing 警示）
應收帳款年均成長 41.8%，遠超營收成長 5.0%（差 36.8ppt）。典型操縱手法：月底大量出貨給關聯通路衝業績，但貨品最終退回；或對空殼客戶開立假發票認列收入。
