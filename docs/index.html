⚖️ 財務健康偵測在看什麼？
  • 盈餘品質：淨利是否有對應現金流？（長期不一致可能反映應計項目問題）
  • 資產與營收比例：資產成長是否與業務規模相符？
  • 應收帳款變化：收款速度是否跟上業務成長？
  • 毛利率穩定性：是否有難以解釋的大幅跳動？
  • 股東權益變動：獲利是否如預期累積於帳面淨值？

📖 DuPont 分解：ROE = 淨利率 × 資產周轉率 × 財務槓桿

  • 淨利率高 → 核心業務獲利能力強（品牌、技術優勢）
  • 資產周轉率高 → 資產使用效率佳
  • 財務槓桿高 → 借款放大報酬（需同時關注償債能力）
